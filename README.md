# ID Card Details

A new Flutter project for a Digital Business Card.

## Project Description
This is a simple Flutter application that displays professional contact details in a clean card format. 

### Features
* **Profile Image:** Shows a user avatar.
* **Personal Info:** Displays Name and Job Title.
* **Contact Details:** Includes Phone number and Email ID using ListTile widgets.

## How to Run
1. Make sure you have Flutter installed.
2. Connect your emulator or physical device.
3. Run `flutter run` in the terminal.
